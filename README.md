🎮 Rock Paper Scissors GUI Game

A fun and interactive Rock-Paper-Scissors desktop game built using Python Tkinter with a modern GUI interface.

This project allows users to play against the computer, tracks scores, and provides an engaging visual experience with a clean interface.

✨ Features

✅ Modern GUI built with Tkinter
✅ Interactive dropdown selection for player moves
✅ Random computer move generation
✅ Real-time result display
✅ Score tracking system
✅ User-friendly interface
✅ Lightweight and beginner-friendly Python project

🖥️ Tech Stack

Python
Tkinter (GUI Framework)
Random Module

🎯 How It Works

User selects:
Rock 🪨
Paper 📄
Scissors ✂️
Computer randomly chooses one option.
The game compares both choices.
Winner is displayed instantly.
Score gets updated.
